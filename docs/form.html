🪣 ITH บันทึกข้อมูล
📊 ไปหน้า Dashboard

ฟอร์มบันทึกข้อมูลการขุดดิน

กรอกข้อมูลด้านล่างเพื่อนำไปแสดงบน Dashboard

📌 ทำไมต้องอัปโหลดไฟล์? (เพราะยังไม่มี Server)

ตอนกรอกฟอร์ม → ข้อมูลเก็บในเครื่องคุณคนเดียว (เหมือนบันทึกในโน้ตส่วนตัว) คนอื่นเปิด Dashboard จะยังไม่เห็น

ตอนอัปโหลด records.json ขึ้น GitHub → ข้อมูลไปอยู่บนเว็บร่วมกัน ทุกคนเปิด Dashboard จะเห็นเหมือนกัน

📖 วิธีอัปโหลดทีละขั้น (คลิกเปิด)
  1. กรอกข้อมูลในฟอร์ม แล้วกด บันทึก ตามปกติ
  2. กดปุ่ม ⬇️ ดาวน์โหลด records.json ด้านล่าง → ไฟล์จะลงเครื่องคุณ
  3. เปิด GitHub → เข้า repo ของโปรเจกต์
  4. ไปที่โฟลเดอร์ docs → data → คลิกไฟล์ records.json
  5. กดปุ่ม ✏️ Edit (แก้ไข)
  6. ลบเนื้อหาเดิมทั้งหมด แล้วเปิดไฟล์ที่ดาวน์โหลดมา คัดลอกวางแทน
  7. กด Commit changes ด้านล่าง
  8. รอ 1–2 นาที แล้ว Refresh หน้า Dashboard — ทุกคนจะเห็นข้อมูลใหม่

📋 รายการที่บันทึกวันนี้ — การขุดดิน

📋 รายการที่บันทึกวันนี้ — Downtime
